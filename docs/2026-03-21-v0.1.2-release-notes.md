# Genesis Way 0.1.2 Release Notes
wa
Release date: 2026-03-21

## Summary
Version 0.1.2 is a quality and polish release focused on readability, in-flow navigation clarity, and motivation-preserving progress cues across the daily flow.

This release does not introduce major new platform features. It hardens the core user experience for Dump -> Shape -> Fill.

## Highlights
- Improved baseline text legibility and contrast in the default Brown Glass theme.
- Added a clear Home return action from in-flow screens.
- Strengthened completion and progress feedback to reinforce momentum.
- Improved Shape guidance readability for Work vs Personal lane decisions.
- Improved app icon configuration reliability in project settings (alternate icon mapping consistency).

## What Changed
### UI and UX improvements
- Increased readability for previously dim support text and micro-copy.
- Added a top-level Home affordance on in-flow screens to reduce navigation friction.
- Added explicit progress feedback in Shape for item readiness.
- Added stronger execution-progress visibility and completed-state cues in Fill.
- Refined Shape guidance copy legibility so lane intent is easier to understand quickly.

### Stability and project configuration
- Fixed Xcode project file parsing issue related to icon build-setting key syntax.
- Added iPad icon dictionary parity for alternate icon configuration consistency.

## Impact
- Faster orientation and lower cognitive friction while moving through the daily loop.
- Better completion visibility for users who rely on motivational cues.
- Safer handoff from development to device testing and release build workflows.

## Known limitations
- Icon artwork centering polish is still being finalized and remains tracked as open polish work.

## Recommended validation for this release
- Run on-device checks for legibility in normal and bright light.
- Confirm Home affordance discoverability from Shape, Fill, and Park.
- Validate Shape and Fill progress/complete-state feedback updates in a mixed task state.
- Validate icon style switching persistence and behavior after relaunch.

## Next up
- 0.2.0: Authentication and user account foundation.
